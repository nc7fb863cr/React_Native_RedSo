# Introduction
* MainScreen
  * 管理三個Team分頁之Navigation
  * 管理Global State，並分派至三個Team分頁
  * 使用ScrollView實現分頁滑動
* Team
  * 透過Flatlist實現下拉更新
  * 透過Card展示橫幅或人員卡片
